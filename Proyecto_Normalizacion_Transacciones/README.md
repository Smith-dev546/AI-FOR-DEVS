# 💰 Sistema de Normalización de Transacciones Multifuente

## 🎯 Descripción

Sistema para normalizar transacciones financieras provenientes de múltiples fuentes con diferentes formatos, permitiendo su exploración y análisis a través de una interfaz interactiva.

## ✨ Características

- 🔄 **Normalización Automática**: Detecta fuente y normaliza transacciones
- 🏷️ **Mapeo Inteligente**: Convierte monedas, estados y formatos de fecha
- ✅ **Validación Robusta**: Identifica transacciones inválidas o incompletas
- 📊 **Métricas Avanzadas**: Estadísticas completas por estado, moneda y fuente
- 🔍 **Filtrado Interactivo**: Filtra transacciones por múltiples criterios
- 📤 **Exportación**: Guarda resultados en formato JSON
