# 📊 Sistema de Análisis de Logs

## 🎯 Descripción

Sistema desarrollado en Python para analizar archivos de logs, identificar líneas válidas e inválidas, y generar estadísticas detalladas sobre la calidad y distribución de eventos.

## ✨ Características

- 📁 Análisis de archivos de log (.txt, .log)
- 🏷️ Identificación de niveles: INFO, WARNING, ERROR
- 📅 Validación de fechas con formato YYYY-MM-DD
- 📊 Generación de estadísticas detalladas
- 📈 Visualización con barras de porcentaje
- 📄 Exportación de reportes
- 🎯 Detección de líneas mal formadas
- 🔍 Análisis detallado línea por línea

### Requisitos

- Sistema operativo: Windows, macOS o Linux
